# MSU Master 1st Semester Courses

**[English](README_EN.md)** | 中文

莫斯科国立大学计算机学院硕士一年级课程资料

## 📚 课程目录

### Java
**Java程序设计** (Программирование на Java)

Java 编程语言课程，由 Romanov 教授主讲。

- `PPT/` - 课程讲义
  - Java 基础介绍
  - Java 编程实践

---

### Kotlin
**Kotlin程序设计** (Программирование на Kotlin)

Kotlin 编程语言课程，由 Romanov 教授主讲。

- `PPT/` - 课程讲义（共10讲）
  - Kotlin 语言介绍与基础语法
  - Kotlin 高级特性

---

### MCMR
**机械臂机器人建模与控制** (Моделирование и управление манипуляционными роботами)

机械臂机器人学课程，涵盖建模、运动学与动力学。

- `PPT/` - 课程讲义
  - 课程介绍 (Intro)
  - 正运动学 (Forward Kinematics)
  - 逆运动学 (Inverse Kinematics)
  - 动力学 (Dynamics)
- `homework/` - 课程作业

---

### MHBG
**异构大数据管理** (Управление разно-структурированными большими данными)

大数据与高性能计算课程。

- `PPT/` - 课程讲义
- `Lab01-MapReduce/` - MapReduce 实验
- `Lab02-Assignment/` - 作业 2

---

### PPSC
**并行编程与超级计算机设计** (Параллельное программирование и суперкомпьютерный кодизайн)

并行计算与超算编程课程。

- `PPT/` - 课程讲义（共7讲）
- `Lab01-HPL/` - HPL (High-Performance Linpack) 基准测试
- `Lab02-Top500/` - Top500 超级计算机分析
- `Lab03-ParallelSTL/` - Parallel STL 并行编程
- `Lab04-SIMD/` - SIMD 内置函数编程
- `Lab05-Assignment/` - 作业 5
- `Lab06-Assignment/` - 作业 6

---

### Science
**科研论文资料**

生成对抗网络 (GAN) 和扩散模型相关论文：

- `Generative Adversarial Nets.pdf` - GAN 原始论文 (Goodfellow et al.)
- `Denoising Diffusion Probabilistic Models.pdf` - DDPM 论文
- `少样本条件下基于生成对抗网络的遥感图像数据增强.pdf` - 遥感图像数据增强
- 俄文论文：GAN 图像生成与音频恢复对比研究

---

### Admin
**课程计划**

- `Учебный план магистры 1 курс ФИИТ.xls` - 硕士一年级课程教学计划
