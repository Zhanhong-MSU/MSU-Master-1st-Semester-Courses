# MSU Master 1st Semester Courses

莫斯科国立大学计算机学院硕士一年级课程资料

## 📚 课程目录

### Java
**Java程序设计** (Программирование на Java)

Java 编程语言课程，由 Romanov 教授主讲。

- `PPT/` - 课程讲义
  - Java 基础介绍
  - Java 编程实践

---

### Kotlin
**Kotlin程序设计** (Программирование на Kotlin)

Kotlin 编程语言课程，由 Romanov 教授主讲。

- `PPT/` - 课程讲义（共10讲）
  - Kotlin 语言介绍与基础语法
  - Kotlin 高级特性

---

### MCMR
**移动和协作机器人的数学与控制** (Математика и Управление Мобильными и Кооперативными Роботами)

机器人学课程，涵盖运动学与动力学。

- `PPT/` - 课程讲义
  - 课程介绍 (Intro)
  - 正运动学 (Forward Kinematics)
  - 逆运动学 (Inverse Kinematics)
  - 动力学 (Dynamics)
- `homework/` - 课程作业

---

### MHBG
**高性能大数据方法** (Методы высокопроизводительной обработки больших данных)

大数据与高性能计算课程。

- `PPT/` - 课程讲义
- `No.1/`, `No.2/` - 课程作业

---

### PPSC
**超级计算机系统并行编程** (Параллельное программирование суперкомпьютерных систем)

并行计算与超算编程课程。

- `PPT/` - 课程讲义（共7讲）
- `No.1/` - HPL (High-Performance Linpack) 基准测试
- `No.2/` - Top500 超级计算机分析
- `No.3/` - Parallel STL 并行编程
- `No.4/` - SIMD 内置函数编程
- `No.5/`, `No.6/` - 其他作业

---

### Science
**科研论文资料**

生成对抗网络 (GAN) 和扩散模型相关论文：

- `Generative Adversarial Nets.pdf` - GAN 原始论文 (Goodfellow et al.)
- `Denoising Diffusion Probabilistic Models.pdf` - DDPM 论文
- `少样本条件下基于生成对抗网络的遥感图像数据增强.pdf` - 遥感图像数据增强
- 俄文论文：GAN 图像生成与音频恢复对比研究

---

### Owo
**课程计划**

- `Учебный план магистры 1 курс ФИИТ.xls` - 硕士一年级课程教学计划
